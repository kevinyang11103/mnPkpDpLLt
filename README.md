# Python博客管理系统 python504

## 项目概述

本项目是基于Python+Django框架开发的博客管理系统。系统提供完整的前后台功能，旨在帮助用户便捷地管理和发布博客内容。

## 技术栈

- **后端：**
  - Python
  - Django
  - Django REST framework（提供API接口）

- **前端：**
  - HTML/CSS/JavaScript
  - Bootstrap

- **数据库：**
  - PostgreSQL

## 功能模块

- **用户管理：**
  - 用户注册、登录、注销
  - 用户信息管理

- **内容管理：**
  - 博客文章的创建、编辑、删除
  - 标签与分类管理

- **系统管理：**
  - 网站设置
  - 日志管理

## 系统角色

- **普通用户：**
  - 可以浏览、评论博客文章

- **管理员：**
  - 拥有系统所有的管理权限

## 运行环境

- **后端：**
  - Python 3.8 或以上版本
  - Django 3.2 或以上版本

- **前端：**
  - 兼容主流现代浏览器

- **数据库：**
  - PostgreSQL 13 或以上版本

## 部署步骤

1. 克隆项目代码到本地
2. 安装依赖项（`pip install -r requirements.txt`）
3. 配置数据库
4. 运行 `python manage.py runserver` 启动开发服务器

- **前后端分离：** 提供API接口，易于前端多种形式的接入。
